## GitHub Actions 自动构建

本项目使用GitHub Actions自动构建Android和iOS应用。

### 构建方式

1. Fork或克隆此仓库到您的GitHub账号
2. 进入GitHub仓库页面，点击"Actions"选项卡
3. 选择需要运行的工作流（Android构建或iOS构建）
4. 点击"Run workflow"按钮，选择分支并启动构建

### 构建产物

- Android构建会生成APK文件
- iOS构建会生成未签名的应用文件

构建完成后，可以在Actions执行记录中下载构建产物。
